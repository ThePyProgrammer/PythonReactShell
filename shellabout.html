The React Shell - Python 3.7.2

An enterprise by Buvan Kirthik s/o Pandian, Prannaya Gupta and Yap Yuan Xi

Try it out:

It's quite simple! You have to do the following:

  1. First, you can input a calculated value. This value will be stored as ans.
  2. Secondly, you can make something a constant by running it in the const_ping function. You have to input whether it should be stored as a string value (if so, input 'str') or a number type (if so, input 'num').
  3. you cannot import modules or use multi-line loops, so you will have to input break() so that the object value will break. Then you can do whatever you want to do!






Pros of this enterprise Cons of this enterprise
It is a user-friendly version of python! You can operate many user-friendly functions and over long distances with functions such as const_ping()! You can't really span multi-line codes. Therefore, it is advised that you make functions in the source code itself.
